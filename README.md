# IDGen - מחולל ובודק ספרות אקראיות

כלי לימודי לחקר ספרת ביקורת - אתר סטטי ב-HTML/CSS/JavaScript

## תכונות

- בודק תקינות רצפי ספרות
- מחולל רצף יחיד
- מחולל קבוצתי (עד 1000 רצפים)
- הסבר מפורט על האלגוריתם
- תמיכה במצב כהה/בהיר
- תמיכה מלאה ב-RTL (עברית)
- רספונסיבי למובייל



## קבצים

- `index.html` - דף הבית
- `validator.html` - בודק תקינות
- `generator.html` - מחולל יחיד
- `batch.html` - מחולל קבוצתי
- `algorithm.html` - הסבר האלגוריתם
- `about.html` - אודות
- `styles.css` - עיצוב מלא
- `app.js` - ניהול ערכת נושא
- `check-digit.js` - לוגיקת האלגוריתם
- `validator.js` - פונקציות בודק תקינות
- `generator.js` - פונקציות מחולל יחיד
- `batch.js` - פונקציות מחולל קבוצתי
